# Retrofit

## Deskripsi
Retrofit adalah library standar untuk komunikasi dengan REST API dalam aplikasi Android. Anda akan belajar bagaimana mengimplementasikan networking menggunakan Retrofit.

## Tujuan Pembelajaran
- Menggunakan Retrofit untuk komunikasi API
- Menerapkan OkHttp dan Interceptors
- Mengelola request dan response

## Materi
- Konfigurasi Retrofit
- Membuat API interface
- OkHttp client
- Interceptors (Authentication, Logging, Headers)
- Error handling
- Pengganti Volley dan AsyncTask
- Standard networking approach

## Catatan
Retrofit adalah standard networking library dalam pengembangan Android saat ini.
