# Permissions Basic

## Deskripsi
Permissions penting untuk menjaga privacy dan security aplikasi. Anda akan belajar tentang runtime permissions yang diperkenalkan sejak Android 6.0.

## Tujuan Pembelajaran
- Memahami konsep permissions di Android
- Mampu meminta runtime permissions
- Menangani response dari permission request

## Materi
- Jenis-jenis permissions (normal, dangerous, signature)
- Runtime permissions vs manifest permissions
- Permission request flow
- Menangani permission response
- Best practices permissions
- Compatibility dengan versi Android lama

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Privacy & security)

## Catatan
Aplikasi Android modern harus mematuhi kebijakan permission untuk diterima di Google Play Store.
