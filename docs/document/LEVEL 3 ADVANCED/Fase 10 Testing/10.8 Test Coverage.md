# Test Coverage

## Deskripsi
Test coverage mengukur seberapa banyak kode yang ditest. Anda akan belajar menggunakan JaCoCo untuk menghasilkan laporan coverage dan metrik kualitas kode.

## Tujuan Pembelajaran
- Menggunakan JaCoCo untuk test coverage
- Menghasilkan coverage reports
- Memahami metrik kualitas kode

## Materi
- Setup JaCoCo dalam proyek
- Coverage reports generation
- Line coverage vs branch coverage
- Coverage thresholds
- Kode tanpa coverage tracking
- Code quality metrics
- Test coverage dalam CI/CD

## Waktu
2 hari

## Prioritas
⭐⭐ (Code quality metrics)

## Catatan
Test coverage metrics penting untuk mengukur dan menjaga kualitas kode dalam jangka panjang.
