# Navigation Component (View-based)

## Deskripsi
Meskipun Compose lebih baru, banyak aplikasi masih menggunakan pendekatan View-based. Anda akan belajar Navigation Component XML dan Safe Args untuk aplikasi legacy.

## Tujuan Pembelajaran
- Menggunakan Navigation Component XML
- Menggunakan Safe Args untuk type safety
- Menangani navigasi dalam aplikasi View-based

## Materi
- Navigation XML
- Safe Args plugin
- NavGraph dan destinations
- Action dan argument dalam XML
- Fragment transactions management
- Untuk legacy View apps
- Migration path dari View ke Compose

## Waktu
4 hari

## Prioritas
⭐⭐ (For legacy View apps)

## Catatan
Meskipun Compose adalah pendekatan modern, pengetahuan tentang View-based navigation masih penting untuk aplikasi legacy.
