# Configuration Changes

## Deskripsi
Configuration changes seperti rotasi layar dapat menyebabkan Activity ter-destroy dan re-created. Anda akan belajar bagaimana menangani perubahan konfigurasi ini dengan baik.

## Tujuan Pembelajaran
- Memahami proses configuration changes
- Menangani rotasi layar
- Menangani multi-window mode

## Materi
- Proses destroy dan create kembali Activity
- onSaveInstanceState dan onRestoreInstanceState
- Konfigurasi lainnya (locale, theme, screen size)
- Menangani multi-window mode
- Alternatif pendekatan state preservation

## Waktu
2 hari

## Prioritas
⭐⭐ (App stability)

## Catatan
Menangani configuration changes dengan baik penting untuk pengalaman pengguna yang konsisten.
