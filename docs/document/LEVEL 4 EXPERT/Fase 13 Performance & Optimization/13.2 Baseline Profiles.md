# Baseline Profiles

## Deskripsi
Baseline Profiles adalah fitur untuk meningkatkan performa aplikasi dengan memberi tahu sistem tentang kode yang sering dieksekusi. Anda akan belajar implementasi dan Macrobenchmark.

## Tujuan Pembelajaran
- Menggunakan Baseline Profiles
- Mengimplementasikan Macrobenchmark
- Menggantikan manual optimization

## Materi
- Konsep Baseline Profiles
- Membuat baseline profile
- Macrobenchmark untuk performa
- Profile-guided optimization
- Performance boost melalui profiles
- Manual optimization problems
- AOT dan JIT optimization dalam Android
- Performance improvement dengan profiles
