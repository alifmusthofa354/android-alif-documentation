# CameraX

## Deskripsi
CameraX adalah library modern untuk mengakses kamera dalam aplikasi Android. Anda akan belajar implementasi CameraX API untuk preview dan image capture.

## Tujuan Pembelajaran
- Menggunakan CameraX API
- Mengimplementasikan Preview
- Menggunakan ImageCapture dalam aplikasi

## Materi
- Konsep CameraX
- Setup Preview use case
- ImageCapture use case
- CameraProvider dan CameraSelector
- Capture mode dan configuration
- Pengganti Camera2 API dan Camera API
- Modern camera implementation
- Camera permissions dan best practices

1 minggu

⭐⭐ (Modern camera)

## Catatan
CameraX menyederhanakan akses kamera dan memberikan kompatibilitas yang baik di berbagai perangkat.
