# Fragments

## Deskripsi
Fragment digunakan untuk membuat UI modular dan fleksibel, terutama dalam aplikasi dengan tampilan kompleks atau multi-panel. Anda akan belajar konsep Fragment dan bagaimana menggunakannya.

## Tujuan Pembelajaran
- Memahami konsep Fragment
- Memahami lifecycle Fragment
- Melakukan fragment transactions

## Materi
- Apa itu Fragment
- Fragment lifecycle
- Fragment vs Activity
- Fragment transactions (add, replace, remove)
- Communication antar fragments
- Fragment arguments
- Best practices

## Waktu
3 hari

## Prioritas
⭐⭐ (Modular UI components)

## Catatan
Fragment sangat berguna untuk membuat UI yang fleksibel, terutama untuk tablet dan layar besar.
