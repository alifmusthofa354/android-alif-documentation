# Navigation Compose

## Deskripsi
Navigation Compose adalah pendekatan type-safe untuk navigasi dalam aplikasi Jetpack Compose. Anda akan belajar bagaimana mengimplementasikan navigasi yang aman dan efisien.

## Tujuan Pembelajaran
- Menggunakan Navigation Compose
- Mengimplementasikan NavHost dan NavController
- Menggantikan Navigation Component XML

## Materi
- Setup Navigation Compose
- NavHostController dan NavController
- Definisi route dalam Compose
- Type-safe navigation
- Passing data antar screens
- Animated navigation
- Navigation Compose benefits

## Waktu
1 minggu

## Prioritas
⭐⭐⭐ (Type-safe navigation)

## Catatan
Navigation Compose memberikan pendekatan yang type-safe dan lebih terintegrasi dengan Compose.
