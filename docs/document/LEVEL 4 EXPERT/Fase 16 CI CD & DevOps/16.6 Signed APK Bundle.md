# Signed APK/Bundle

## Deskripsi
Signing aplikasi adalah kewajiban untuk distribusi di Play Store. Anda akan belajar keystore management dan signing configuration untuk production release.

## Tujuan Pembelajaran
- Mengelola keystore untuk signing
- Mengkonfigurasi signing dalam build
- Menggantikan unsigned builds

## Materi
- Keystore management dalam Android
- Signing configuration dalam Gradle
- Release signing process
- APK vs App Bundle signing
- Unsigned builds problems
- Production release requirements
- Security dari signed applications

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Production release)

## Catatan
Signed APK/Bundle adalah keharusan untuk distribusi aplikasi secara resmi di Google Play Store.
