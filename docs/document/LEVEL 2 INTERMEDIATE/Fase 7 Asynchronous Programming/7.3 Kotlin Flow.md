# Kotlin Flow

## Deskripsi
Flow adalah bagian dari coroutines yang digunakan untuk mengelola stream data secara asynchronous. Anda akan belajar StateFlow dan SharedFlow sebagai pengganti LiveData.

## Tujuan Pembelajaran
- Memahami konsep Kotlin Flow
- Menggunakan StateFlow untuk state management
- Menggunakan SharedFlow untuk event broadcasting

## Materi
- Konsep Flow sebagai reactive stream
- StateFlow vs SharedFlow
- StateFlow untuk state observation
- SharedFlow untuk event
- Pengganti LiveData dan RxJava
- Reactive streams dalam Android
- Integration dengan ViewModels

## Catatan
