# Bluetooth & NFC

## Deskripsi
Bluetooth LE dan NFC API memungkinkan komunikasi dengan perangkat dan kartu di sekitar. Anda akan belajar implementasi komunikasi perangkat dalam aplikasi Android.

## Tujuan Pembelajaran
- Menggunakan Bluetooth LE untuk komunikasi perangkat
- Mengimplementasikan NFC API
- Menggantikan pendekatan Bluetooth klasik

## Materi
- Bluetooth LE (Low Energy) implementation
- NFC API usage
- Pairing dan connection management
- Data transfer melalui Bluetooth
- NFC tag reading dan writing
- Classic Bluetooth problems
- Device communication dalam aplikasi
