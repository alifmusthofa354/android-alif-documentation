# Compose Testing

## Deskripsi
Compose testing menggunakan pendekatan yang berbeda dari Espresso. Anda akan belajar Compose test rules dan semantics untuk testing UI Compose.

## Tujuan Pembelajaran
- Menggunakan Compose test rules
- Memahami semantics dalam testing
- Menggantikan pendekatan Espresso untuk Compose

## Materi
- ComposeTestRule dan testing mechanics
- Semantics dan semantic actions
- Testing Composable functions
- Interaction dengan UI dalam testing
- Pengganti Espresso untuk Compose
- UI testing dalam Jetpack Compose
- Testing composables secara efektif

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (UI testing)

## Catatan
Compose testing memberikan pendekatan yang lebih efisien dan type-safe untuk testing UI dalam Compose.
