# Terminal/Command Line

## Deskripsi
Penggunaan terminal atau command line sangat penting dalam workflow pengembangan software. Anda akan belajar navigasi dasar dan perintah-perintah penting di command line.

## Tujuan Pembelajaran
- Memahami penggunaan terminal/command line
- Mampu melakukan navigasi file dan folder
- Menggunakan perintah dasar untuk manajemen file

## Materi
- Pengenalan terminal/command prompt
- Navigasi direktori (cd, ls, dir)
- Manipulasi file dan folder (mkdir, touch, rm, cp, mv)
- Membaca path dan permission
- Perintah dasar di Windows (cmd) dan Unix/Linux/Mac (bash)
- Penggunaan dalam workflow development

## Waktu
2 hari

## Prioritas
⭐⭐ (Developer workflow)

## Catatan
Pemahaman tentang terminal akan mempercepat workflow pengembangan Anda.
