# Compose Fundamentals

## Deskripsi
Jetpack Compose adalah toolkit modern untuk membangun UI Android. Anda akan belajar dasar-dasar Compose dan mengapa ini merupakan masa depan UI Android.

## Tujuan Pembelajaran
- Memahami konsep declarative UI
- Mampu membuat @Composable functions
- Membandingkan Compose dengan pendekatan XML

## Materi
- Konsep declarative UI vs imperative UI
- @Composable functions
- State dan mutableStateOf
- Preview functions
- Keunggulan Jetpack Compose
- Menggantikan XML layouts
