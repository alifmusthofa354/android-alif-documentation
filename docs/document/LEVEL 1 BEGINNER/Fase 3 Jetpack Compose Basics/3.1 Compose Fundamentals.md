## 1Ô∏è‚É£ Pengenalan Jetpack Compose üåü

### Konsep Dasar
Jetpack Compose seperti revolusi dalam dunia UI Android üî• - pendekatan deklaratif modern yang menggantikan XML dan View system tradisional!

### Apa itu Jetpack Compose?
Jetpack Compose adalah toolkit modern UI dari Google untuk membangun aplikasi Android. Ini adalah toolkit deklaratif yang memungkinkan kamu membangun antarmuka pengguna dengan cepat menggunakan Kotlin secara eksklusif.

### Keunggulan Jetpack Compose:

#### 1. **Kurangi Jumlah Kode**
```kotlin
// Dengan XML dan Views (lama)
// activity_main.xml
/*
<TextView
    android:id="@+id/textView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello World!" />
*/

// MainActivity.java
// TextView textView = findViewById(R.id.textView);
// textView.setText("Hello World!");
```

```kotlin
// Dengan Jetpack Compose (baru)
@Composable
fun Greeting(name: String) {
    Text(text = "Hello $name!")
}
```

#### 2. **Pendekatan Deklaratif**
Daripada menjelaskan *bagaimana* UI harus dibangun (imperatif), kamu menjelaskan *apa* yang kamu ingin tampilkan (deklaratif).

#### 3. **Integrasi Penuh dengan Kotlin**
- Tidak perlu XML
- Penggunaan fitur Kotlin secara penuh (coroutines, DSL, dll)
- Type safety

### Perbedaan Declarative vs Imperative UI:

#### Imperative UI (Cara Lama):
```kotlin
// Kamu memberi instruksi LANGKAH DEMI LANGKAH
val button = Button(this)
button.setText("Klik Saya")
button.setOnClickListener { /* handle click */ }
container.addView(button)
```

#### Declarative UI (Compose):
```kotlin
// Kamu mendeklarasikan APA yang kamu inginkan
@Composable
fun MyButton() {
    Button(onClick = { /* handle click */ }) {
        Text("Klik Saya")
    }
}
```

## 2Ô∏è‚É£ @Composable Functions ‚ö°

### Konsep Dasar
@Composable Functions seperti koki ajaib ‚ú® - menerima bahan (parameter) dan menghasilkan UI yang cantik!

### Syntax dan Struktur:
```kotlin
@Composable
fun NamaKomponen(parameter: Tipe) {
    // Fungsi ini menggambarkan bagian dari UI
    Text(text = "Ini adalah komponen")
}
```

### Naming Convention:
- Gunakan **PascalCase** untuk nama fungsi composable
- Nama harus deskriptif
- Hindari prefiks seperti "create", "build", dll

```kotlin
// ‚úÖ Benar
@Composable
fun UserProfileCard() { }

@Composable
fun ProductListItem(product: Product) { }

// ‚ùå Salah
@Composable
fun createUserProfile() { }

@Composable
fun build_product_list() { }
```

### Parameter dan Return Type:
```kotlin
// Parameter dengan tipe jelas

    Card {
        Column {

            Text("Usia: $age")

) {
    Button(
        onClick = onClick,
        enabled = enabled
    ) {

// Unit adalah default return type (tidak perlu ditulis)
@Composable
fun MyComposable(): Unit {  // Unit bisa dihilangkan
    Text("Contoh dengan Unit")
}
```

### Composable Function Rules:

#### 1. **Hanya Panggil dari Composable Context**
```kotlin
@Composable
fun ValidComposable() {
    Text("Ini valid")  // ‚úÖ OK - dipanggil dari composable
    AnotherComposable()  // ‚úÖ OK - dipanggil dari composable
}

fun InvalidComposable() {
    Text("Ini tidak valid")  // ‚ùå ERROR - bukan dari composable
    AnotherComposable()  // ‚ùå ERROR - bukan dari composable
}
```

#### 2. **No Side Effects (kecuali secara sengaja)**
```kotlin
// ‚ùå Salah - side effect
@Composable
fun BadComposable() {
    println("Log pemanggilan")  // Side effect
    Thread.sleep(1000)  // Side effect
}

// ‚úÖ Benar - pure composable
@Composable
fun GoodComposable(counter: Int) {
    Text("Counter: $counter")  // Hanya render, tidak ada efek samping
}
```

#### 3. **Dipanggil Secara Berulang (Free Refro Recomposition)**
```kotlin
@Composable
fun ExpensiveCalculation() {
    // ‚ùå JANGAN LAKUKAN INI
    val result = performComplexCalculation()  // Dapat dipanggil ulang tanpa alasan
    Text("Hasil: $result")
}

@Composable
fun EfficientCalculation() {
    // ‚úÖ Gunakan remember untuk menghindari perhitungan berulang
    val result by remember { performComplexCalculation() }
    Text("Hasil: $result")
}
```

### üéì Penjelasan Konsep:
**@Composable Functions seperti Koki Ajaib:**
- `@Composable`: Annotation yang menandakan fungsi ini bisa menggambarkan UI
- **PascalCase**: Nama fungsi harus mengikuti konvensi
- **Parameter**: Input untuk menentukan bagaimana UI akan terlihat
- **Return**: Tidak perlu return value, hanya menggambarkan UI
- **Aturan Fungsi**: Bebas efek samping, bisa dipanggil ulang, hanya dari konteks composable

**Golden Rules:**
1. Hanya dipanggil dari konteks composable
2. Hindari side effects
3. Harus bisa dipanggil ulang (free from side effects)
4. Gunakan konvensi penamaan yang benar

### 1. Composition
Proses pertama kali UI digambar. Ini terjadi saat:
- Composable pertama kali dipanggil
- Bagian UI pertama kali muncul di layar

```kotlin
@Composable
fun CompositionExample() {
    println("1. Ini dicetak saat COMPOSITION")  // Dicetak 1x saat pertama kali
    
    Text("Contoh Composition")
}
```

### 2. Recomposition
Proses memperbarui UI saat data berubah. Ini terjadi saat:
- State berubah
- Parameter fungsi berubah
- Dependency state berubah

```kotlin
@Composable
fun RecompositionExample() {
    var counter by remember { mutableStateOf(0) }
    
    println("2. Ini DICETAK SAAT RECOMPOSITION (count: $counter)")
    
    Column {
        Text("Counter: $counter")
        Button(onClick = { counter++ }) {
            Text("Tambah")
        }
    }
}
```

### 3. Kapan Recomposition Terjadi?

#### a. State Berubah
```kotlin
@Composable
fun StateRecomposition() {
    var name by remember { mutableStateOf("Ahmad") }

        Text("Ubah Nama")
    }

```

#### b. Parameter Berubah
```kotlin
@Composable
fun ParameterRecomposition(user: User) {
    Text("User: ${user.name}")  // Akan direcompose saat user berubah
}

// Saat user diaktifkan/deaktivasi, komponen ini direcompose
```

### Smart Recomposition
Compose pintar - hanya merecompose bagian yang benar-benar perlu:

```kotlin
@Composable
fun SmartRecompositionDemo() {
    var counter by remember { mutableStateOf(0) }
    val expensiveValue = performExpensiveOperation()  // Ini hanya dihitung sekali
    
    Column {
        // Hanya bagian ini yang direcompose saat counter berubah
        Text("Counter: $counter")
        
        // Bagian ini TIDAK direcompose saat counter berubah
        Text("Expensive: $expensiveValue")
        
        Button(onClick = { counter++ }) {
            Text("Tambah Counter")
        }
    }
}
```

### Skipping Recomposition
Compose secara otomatis melewati recomposition jika nilainya tidak berubah:

```kotlin
@Composable
fun SkippingDemo() {

    var counter by remember { mutableStateOf(0) }

        Text("Text: $text")
        
        // Akan direcompose hanya saat counter berubah
        Text("Counter: $counter")
        
        Button(onClick = { text = "World" }) {
            Text("Ubah Text")
        }
        
        Button(onClick = { counter++ }) {
```

### üéì Penjelasan Konsep:
**Compose Lifecycle seperti Tumbuh Kembang Tanaman:**
- **Composition**: Saat tanaman pertama kali ditanam
- **Recomposition**: Saat tanaman tumbuh/menyesuaikan terhadap lingkungan
- **Decommission**: Saat tanaman dihilangkan

**Smart Recomposition:**
- Compose pintar: hanya merecompose yang perlu
- Melewatkan yang tidak berubah (skipping)
- Mengoptimalkan performa secara otomatis

**Efek Samping (Side Effects):**
- Gunakan `LaunchedEffect` untuk efek samping
- Gunakan `DisposableEffect` untuk membersihkan
- Hindari efek samping dalam komposisi biasa

```kotlin
private val DarkColorScheme = darkColorScheme(

    tertiary = Pink80

    primary = Purple40,
    secondary = PurpleGrey40,
    tertiary = Pink40

    surface = Color(0xFFFFFBFE),
    onPrimary = Color.White,

    onSurface = Color(0xFF1C1B1F),

)

@Composable
fun MyComposeAppTheme(
    darkTheme: Boolean = isSystemInDarkTheme(),
    // Dynamic color is available on Android 12+
    dynamicColor: Boolean = true,
    content: @Composable () -> Unit
) {
    val colorScheme = when {
        dynamicColor && android.os.Build.VERSION.SDK_INT >= android.os.Build.VERSION_CODES.S -> {
            val context = androidx.compose.ui.platform.LocalContext.current
            if (darkTheme) dynamicDarkColorScheme(context) else dynamicLightColorScheme(context)

        else -> LightColorScheme

        colorScheme = colorScheme,
        typography = Typography,
        content = content
    )
```

### Konsep Dasar
Debugging Compose seperti detektif üîç - mencari dan memperbaiki masalah dalam komponen!

### 1. Layout Inspector untuk Compose:
Android Studio menyediakan **Layout Inspector** yang bisa menangani Compose UI:

#### Cara Menggunakannya:
1. Jalankan aplikasi di device/emulator
2. Buka **View > Tool Windows > Layout Inspector**
3. Pilih sesi aplikasi
4. Inspeksi hirarki Compose

### 2. Recomposition Counts:
Untuk melihat berapa kali komponen direcompose:

```kotlin
@Composable
fun MonitoredComposable(value: String) {
    // Tambahkan counter untuk menghitung recomposition
    var recompositionCount by remember { mutableStateOf(0) }
    recompositionCount++
    
    println("Recomposition count for MonitoredComposable: $recompositionCount")
    
    Text("Value: $value, Recompositions: $recompositionCount")
}
```

### 3. Performance Monitoring:
Gunakan **Performance Monitor** di Android Studio untuk:

#### a. Memory Usage:
- Monitor penggunaan memory saat recomposition
- Identifikasi memory leak

#### b. Frame Rate:
- Monitor FPS untuk UI yang smooth
- Identifikasi frame drops

### 4. Debugging Tools:

#### a. CompositionLocal untuk Debugging:
```kotlin
val LocalDebugInfo = staticCompositionLocalOf { DebugInfo() }
```

```kotlin
@Composable
fun DebuggingDemoScreen() {
    var debugMode by remember { mutableStateOf(false) }
    var selectedItem by remember { mutableStateOf(0) }
    
    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(16.dp),
        verticalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        Text(
            text = "Debugging Demo",
            fontSize = 24.sp,
            fontWeight = FontWeight.Bold
        )
        
        // Debug mode toggle
        Row(
            verticalAlignment = Alignment.CenterVertically
        ) {
            Text("Debug Mode: ", fontWeight = FontWeight.Medium)
            Switch(
                checked = debugMode,
                onCheckedChange = { debugMode = it }
            )
        }
        
        // Recomposition counter
        var recompositionCount by remember { mutableStateOf(0) }
        recompositionCount++
        
        Text(
            text = "Recomposition Count: $recompositionCount",
            color = if (recompositionCount > 1) Color.Red else Color.Green,
            fontWeight = FontWeight.Bold
        )
        
        // Component with potential issues
        if (debugMode) {
            DebugInformation(recompositionCount)
        }
        
        // Sample list with selection
        LazyColumn(
            modifier = Modifier
                .fillMaxWidth()
                .weight(1f)
        ) {
            items(sampleData) { item ->
                DebuggableListItem(
                    item = item,
                    isSelected = selectedItem == item.id,
                    onClick = { selectedItem = item.id },
                    showDebugInfo = debugMode
                )
            }
        }
        
        // Selected item info
        Card(
            modifier = Modifier.fillMaxWidth()
        ) {
            Column(
                modifier = Modifier.padding(16.dp)
            ) {
                Text(
                    text = "Selected Item Info",
                    fontWeight = FontWeight.Bold
                )
                Text("ID: ${sampleData.find { it.id == selectedItem }?.id ?: "None"}")
                Text("Name: ${sampleData.find { it.id == selectedItem }?.name ?: "None"}")
            }
        }
    }
}

@Composable
fun DebugInformation(recompositionCount: Int) {
    Card(
        modifier = Modifier.fillMaxWidth(),
        colors = CardDefaults.cardColors(
            containerColor = Color.Yellow.copy(alpha = 0.3f)
        )
    ) {
        Column(
            modifier = Modifier.padding(16.dp)
        ) {
            Text(
                text = "DEBUG INFO",
                fontWeight = FontWeight.Bold,
                color = Color.Red
            )
            Text("This component is in debug mode")
            Text("Current recomposition count: $recompositionCount")
        }
    }
}

@Composable
fun DebuggableListItem(
    item: SampleItem,
    isSelected: Boolean,
    onClick: () -> Unit,
    showDebugInfo: Boolean
) {
    var itemRenderCount by remember(item.id) { mutableStateOf(0) }
    itemRenderCount++
    
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .padding(4.dp)
            .then(
                if (showDebugInfo) {
                    Modifier.background(Color.Cyan.copy(alpha = 0.1f))
                } else {
                    Modifier
                }
            )
            .then(
                if (isSelected) {
                    Modifier.background(MaterialTheme.colorScheme.primaryContainer)
                } else {
                    Modifier
                }
            ),
        colors = CardDefaults.cardColors(
            containerColor = if (isSelected) MaterialTheme.colorScheme.primaryContainer else MaterialTheme.colorScheme.surface
        ),
        onClick = onClick
    ) {
        Row(
            modifier = Modifier
                .fillMaxWidth()
                .padding(12.dp),
            verticalAlignment = Alignment.CenterVertically,
            horizontalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            Icon(
                imageVector = if (isSelected) Icons.Default.Check else Icons.Default.CheckBoxOutlineBlank,
                contentDescription = if (isSelected) "Selected" else "Not selected",
                tint = if (isSelected) MaterialTheme.colorScheme.primary else MaterialTheme.colorScheme.onSurface
            )
            
            Column(
                modifier = Modifier.weight(1f)
            ) {
                Text(
                    text = item.name,
                    fontWeight = FontWeight.Medium
                )
                Text(
                    text = item.description,
                    fontSize = 12.sp,
                    color = MaterialTheme.colorScheme.onSurfaceVariant
                )
            }
            
            if (showDebugInfo) {
                Text(
                    text = "R: $itemRenderCount",
                    fontSize = 10.sp,
                    color = Color.Red,
                    fontWeight = FontWeight.Bold
                )
            }
        }
    }
}
```
