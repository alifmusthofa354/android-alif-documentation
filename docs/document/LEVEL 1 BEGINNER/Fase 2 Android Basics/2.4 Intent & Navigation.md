# Intent & Navigation

## Deskripsi
Intent digunakan untuk navigasi antar komponen dalam aplikasi Android. Anda akan belajar tentang berbagai jenis intent dan cara menggunakannya.

## Tujuan Pembelajaran
- Memahami konsep Intent
- Mampu menggunakan explicit dan implicit intent
- Melakukan navigasi antar Activity

## Materi
- Jenis-jenis Intent (Explicit dan Implicit)
- startActivity
- Menyertakan data dalam Intent
- Intent filter
- Deep linking (dasar)
- Passing data antar Activity

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Navigasi antar screen)

## Catatan
Kemampuan navigasi sangat penting dalam pengembangan aplikasi Android.
