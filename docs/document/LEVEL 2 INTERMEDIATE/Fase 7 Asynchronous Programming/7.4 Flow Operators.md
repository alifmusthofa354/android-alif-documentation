# Flow Operators

## Deskripsi
Flow operators memungkinkan Anda mentransformasi, memfilter, dan menggabungkan stream data. Anda akan belajar berbagai operator yang tersedia dalam Flow.

## Tujuan Pembelajaran
- Menggunakan operator map, filter, dan combine
- Menggunakan collectLatest untuk efisiensi
- Memahami perbedaan dengan RxJava operators

## Materi
- map, filter, take, drop
- combine, merge, zip
- debounce, throttle
- collectLatest
- distinctUntilChanged
- Pengganti RxJava operators
- Data transformation techniques

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (Data transformation)

## Catatan
Flow operators menyediakan cara deklaratif untuk memanipulasi stream data.
