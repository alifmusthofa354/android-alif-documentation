# Expedited Jobs

## Deskripsi
Expedited WorkManager memungkinkan background work untuk dijalankan dengan prioritas tinggi. Anda akan belajar bagaimana mengimplementasikan expedited jobs untuk Android 12+.

## Tujuan Pembelajaran
- Menggunakan Expedited WorkManager
- Memahami perbedaan dengan regular work
- Menggantikan pendekatan dengan long delays

## Materi
- Expedited WorkManager untuk Android 12+
- Priority dan execution dalam expedited work
- Use cases untuk expedited work
- Perbedaan dengan regular background work
- Fast background work execution
- Expedited work constraints
- Android 12+ specific features

## Waktu
2 hari

## Prioritas
⭐⭐ (Fast background work)

## Catatan
Expedited jobs cocok untuk task yang perlu dijalankan segera tanpa delay.
