# Navigation State Restoration

## Deskripsi
State restoration dalam navigasi kompleks melibatkan multi-backstack dan SaveableStateHolder. Anda akan belajar bagaimana mengelola state saat navigasi kompleks.

## Tujuan Pembelajaran
- Menggunakan multi-backstack navigation
- Menggunakan SaveableStateHolder
- Menangani re-entry dalam navigasi

## Materi
- Multi-backstack dalam Navigation Compose
- SaveableStateHolder
- State restoration dalam navigasi kompleks
- Saveable state dalam screens
- Re-entry handling
- Complex navigation patterns
- State preservation melalui configuration changes

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Complex navigation)

## Catatan
Navigation state restoration penting untuk pengalaman pengguna yang konsisten dalam aplikasi kompleks.
