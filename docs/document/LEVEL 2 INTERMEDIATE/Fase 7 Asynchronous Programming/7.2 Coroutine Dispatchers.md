# Coroutine Dispatchers

## Deskripsi
Dispatcher menentukan thread mana yang digunakan oleh coroutine. Anda akan belajar berbagai dispatcher dan kapan menggunakannya.

## Tujuan Pembelajaran
- Memahami Dispatchers.Main, IO, dan Default
- Menggunakan dispatcher yang sesuai untuk jenis operasi
- Menggantikan manual thread pools

## Materi
- Dispatchers.Main untuk UI operations
- Dispatchers.IO untuk disk/network operations
- Dispatchers.Default untuk CPU-intensive operations
- Custom dispatchers
- Thread management dengan dispatcher
- Performance considerations

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Thread management)

## Catatan
Penggunaan dispatcher yang benar penting untuk performa dan responsivitas aplikasi.
