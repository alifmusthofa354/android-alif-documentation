## **3.2 Basic Composables** (4 hari) ⭐⭐⭐

### **Teknologi Modern:**
- Text, Image, Button
- Column, Row, Box

### **Teknologi yang Ditinggalkan:**
- TextView, ImageView
- LinearLayout

### **Sub-topik Detail:**

#### 1. Text Composable
- Basic Text
- Text styling (fontSize, fontWeight, color)
- TextStyle dan typography
- Text overflow dan maxLines
- TextAlign dan textDecoration
- Annotated strings
- Selectable text

#### 2. Image Composable
- Menampilkan image dari resources (painterResource)
- ContentScale (Crop, Fit, FillWidth, etc)
- ContentDescription untuk accessibility
- Placeholder dan error handling
- Coil library untuk load image dari URL
- Circular/rounded images dengan clip

#### 3. Button Composables
- Button (filled)
- OutlinedButton
- TextButton
- ElevatedButton
- FilledTonalButton
- IconButton
- onClick handler
- enabled/disabled state
- Custom button colors

#### 4. Column Layout
- Vertical arrangement (Top, Center, Bottom, SpaceBetween, SpaceAround, SpaceEvenly)
- Horizontal alignment (Start, CenterHorizontally, End)
- Scroll dengan verticalScroll()
- Nested columns

#### 5. Row Layout
- Horizontal arrangement
- Vertical alignment
- Scroll dengan horizontalScroll()
- Nested rows

#### 6. Box Layout
- Overlay/stacking elements
- Alignment dalam Box
- Z-order (urutan deklarasi)
- matchParentSize vs fillMaxSize
- Use cases: badges, overlays, backgrounds

#### 7. Surface
- Material surface dengan elevation
- Background color
- Border
- Shape (RoundedCornerShape)
- Click handling
