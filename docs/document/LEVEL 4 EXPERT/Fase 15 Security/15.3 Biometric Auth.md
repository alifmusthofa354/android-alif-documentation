# Biometric Auth

## Deskripsi
Biometric authentication memberikan pengalaman login yang aman dan nyaman. Anda akan belajar BiometricPrompt API untuk otentikasi biometrik.

## Tujuan Pembelajaran
- Menggunakan BiometricPrompt API
- Menggantikan pendekatan Fingerprint API lama
- Mengimplementasikan modern authentication

## Materi>
- BiometricPrompt API usage
- Fingerprint, face, dan iris recognition
- Backup authentication methods
- Fingerprint API (deprecated) problems
- Modern authentication dalam aplikasi
- Security dan convenience dalam auth
- Biometric authentication implementation

## Catatan
Biometric auth memberikan keseimbangan antara keamanan dan kenyamanan untuk pengguna.
