# Unit Testing

## Deskripsi
Unit testing adalah dasar dari kualitas perangkat lunak. Anda akan belajar bagaimana melakukan unit testing dalam Kotlin dengan JUnit 5 dan Truth assertions.

## Tujuan Pembelajaran
- Menggunakan JUnit 5 untuk unit testing
- Menggunakan Truth assertions
- Menggantikan JUnit 4

## Materi
- JUnit 5 setup dan struktur
- Truth assertions untuk testing lebih baik
- Unit test structure
- Test lifecycle dan annotations
- Pengganti JUnit 4
- Quality assurance melalui testing
- Best practices dalam unit testing
