# REST API Basics

## Deskripsi
REST API adalah standar untuk komunikasi antara aplikasi client dan server. Anda akan belajar dasar-dasar REST API untuk digunakan dalam aplikasi Android.

## Tujuan Pembelajaran
- Memahami konsep REST API
- Menggunakan HTTP methods dan JSON
- Menerapkan REST principles dalam aplikasi

## Materi
- Konsep REST API
- HTTP methods (GET, POST, PUT, DELETE)
- JSON format
- REST principles
- Perbedaan dengan SOAP
- Understanding APIs dalam konteks Android

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Understanding APIs)

## Catatan
Pemahaman REST API adalah fundamental untuk aplikasi yang berkomunikasi dengan server.
