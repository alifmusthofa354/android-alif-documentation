# Activity & Lifecycle

## Deskripsi
Activity adalah komponen utama dalam aplikasi Android. Anda akan belajar tentang Activity dan lifecycle-nya yang penting untuk manajemen sumber daya aplikasi.

## Tujuan Pembelajaran
- Memahami konsep Activity
- Mengetahui lifecycle Activity
- Mampu menangani savedInstanceState

## Materi
- Apa itu Activity
- Activity lifecycle (onCreate, onStart, onResume, onPause, onStop, onDestroy)
- onSaveInstanceState dan onRestoreInstanceState
- Activity stack
- Transisi lifecycle dalam berbagai kondisi

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Core concept)

## Catatan
Pemahaman lifecycle sangat penting untuk menghindari memory leak dan bug terkait state.
