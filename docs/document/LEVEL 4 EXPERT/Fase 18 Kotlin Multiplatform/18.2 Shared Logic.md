# Shared Logic

## Deskripsi
Berbagi logika bisnis antar platform adalah keuntungan utama dari KMP. Anda akan belajar mengimplementasikan ViewModels dan repository yang bisa digunakan di beberapa platform.

## Tujuan Pembelajaran
- Membagi logika bisnis ke shared module
- Menggunakan ViewModels bersama antar platform
- Menggantikan pendekatan duplicate business logic

## Materi
- Shared ViewModels dalam KMP
- Shared repositories pattern
- Business logic dalam common module
- Code sharing strategies
- Duplicate business logic problems
- Code reuse dalam KMP
- Architecture dalam multiplatform
