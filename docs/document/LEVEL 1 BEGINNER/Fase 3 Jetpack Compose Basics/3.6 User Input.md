# User Input

## Deskripsi
Mengelola input dari pengguna adalah bagian penting dari aplikasi. Dalam Compose, ada pendekatan yang berbeda untuk menangani input pengguna.

## Tujuan Pembelajaran
- Menggunakan TextField untuk input teks
- Menangani klik dan interaksi lainnya
- Membuat UI yang responsif terhadap input pengguna

## Materi
- TextField dalam Compose
- onClick handlers
- Checkbox, Switch, RadioButton
- InteractionState
- Pengganti EditText dan Button XML
- Input validation dasar

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Interactivity)

## Catatan
Kemampuan menangani input pengguna adalah dasar dari aplikasi interaktif.
