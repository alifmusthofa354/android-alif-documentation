# Resources & Assets

## Deskripsi
Resources dan assets adalah tempat menyimpan file-file non-code seperti string, warna, dimensi, gambar, dan lainnya. Anda akan belajar cara menggunakan resources dengan benar.

## Tujuan Pembelajaran
- Memahami struktur resources
- Menggunakan resources alih-alih hardcoded values
- Mengelola berbagai jenis resources

## Materi
- Folder res/
- strings.xml (i18n)
- colors.xml
- dimens.xml
- drawable (gambar dan shape)
- values (styles.xml, themes.xml)
- Assets folder untuk file-file lain
- Best practices penggunaan resources

## Waktu
2 hari

## Prioritas
⭐⭐ (Best practices)

## Catatan
Menggunakan resources adalah best practice untuk memudahkan maintenance dan localization.
