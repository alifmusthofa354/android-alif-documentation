# DataStore

## Deskripsi
DataStore adalah solusi modern dari Google untuk menyimpan data key-value dan typed objects secara async dan konsisten. Anda akan belajar menggantikan SharedPreferences dengan DataStore.

## Tujuan Pembelajaran
- Menggunakan Preferences DataStore
- Menggunakan Proto DataStore
- Memahami keunggulan DataStore dari SharedPreferences

## Materi
- Perbedaan DataStore dan SharedPreferences
- Setup Preferences DataStore
- Penggunaan DataStore dalam aplikasi
- Proto DataStore untuk typed objects
- Error handling dalam DataStore
- Modern key-value storage approach

4 hari

## Prioritas
⭐⭐⭐ (Modern key-value storage)

## Catatan
DataStore adalah pengganti modern untuk SharedPreferences yang lebih aman dan asinkron.
