# Scaffold & AppBar

## Deskripsi
Scaffold menyediakan struktur dasar untuk layar aplikasi Android sesuai panduan Material Design. Anda akan belajar cara menggunakannya beserta TopAppBar dan BottomAppBar.

## Tujuan Pembelajaran
- Menggunakan Scaffold untuk struktur layar
- Mengimplementasikan TopAppBar
- Menggunakan BottomAppBar

## Materi
- Scaffold composable
- TopAppBar
- BottomAppBar
- FloatingActionButton dalam Scaffold
- SnackbarHost dalam Scaffold
- Pengganti dari custom app bar implementation
- Best practices struktur layar

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Standard screen structure)

## Catatan
Scaffold menyediakan struktur standar yang memudahkan implementasi Material Design.
