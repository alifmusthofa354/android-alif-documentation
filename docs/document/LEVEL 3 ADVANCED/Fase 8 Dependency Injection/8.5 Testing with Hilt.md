# Testing with Hilt

## Deskripsi
Testing dengan Hilt memungkinkan Anda mengganti dependensi dengan mock saat testing. Anda akan belajar bagaimana mengatur dan menggunakan Hilt dalam testing.

## Tujuan Pembelajaran
- Menggunakan @HiltAndroidTest untuk testing
- Membuat test modules
- Menggantikan pendekatan manual di testing

## Materi
- @HiltAndroidTest untuk UI testing
- Test modules dalam Hilt
- @UninstallModules untuk mengganti dependensi
- Dependency substitution dalam testing
- Testing dengan fake dependencies
- Testable code melalui Hilt
- Integration testing dengan Hilt

## Waktu
3 hari

## Prioritas
⭐⭐ (Testable code)

## Catatan
Testing dengan Hilt membuat kode lebih mudah diuji dan memungkinkan substitusi dependensi.
