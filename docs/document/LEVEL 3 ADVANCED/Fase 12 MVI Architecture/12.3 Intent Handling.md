# Intent Handling

## Deskripsi
Intent dalam MVI adalah input dari user atau sistem. Anda akan belajar bagaimana menangani user intents dan memproses action dalam sistem MVI.

## Tujuan Pembelajaran
- Memahami user intents dalam MVI
- Mengelola action processing
- Menggantikan pendekatan multiple LiveData

## Materi
- User intents sebagai input
- Action processing dalam MVI
- Intent ke State transformation
- Single source untuk event handling
- Multiple LiveData problems
- Event handling dalam MVI
- Intent pipeline dalam arsitektur
- State transition dari intent
