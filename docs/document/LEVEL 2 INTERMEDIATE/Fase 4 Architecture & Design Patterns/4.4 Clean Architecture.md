# Clean Architecture

## Deskripsi
Clean Architecture adalah pendekatan arsitektur yang memisahkan concerns ke dalam layer-layer yang berbeda. Anda akan belajar bagaimana mengimplementasikan arsitektur ini untuk aplikasi Android yang skalabel.

## Tujuan Pembelajaran
- Memahami konsep Clean Architecture
- Memisahkan aplikasi ke dalam layer: Presentation, Domain, Data
- Mengimplementasikan arsitektur ini dalam proyek Android

## Materi
- Konsep Clean Architecture
- Layer: Presentation, Domain, Data
- Dependency rules dalam Clean Architecture
- Implementasi dalam konteks Android
- Keunggulan dari Monolithic architecture
- Scalability dan testability

## Waktu
1 minggu

## Prioritas
⭐⭐⭐ (Scalability)

## Catatan
Clean Architecture memberikan skalabilitas dan kemudahan dalam maintenance kode.
