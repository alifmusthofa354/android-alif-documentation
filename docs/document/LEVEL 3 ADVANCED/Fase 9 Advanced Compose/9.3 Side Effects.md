# Side Effects

## Deskripsi
Side effects dalam Compose digunakan untuk menyinkronkan efek dengan lifecycle composition. Anda akan belajar LaunchedEffect, DisposableEffect, dan SideEffect.

## Tujuan Pembelajaran
- Menggunakan LaunchedEffect untuk efek async
- Menggunakan DisposableEffect untuk efek yang perlu cleanup
- Menggunakan SideEffect untuk menyinkronkan ke external state

## Materi
- LaunchedEffect untuk efek async dalam composition
- DisposableEffect untuk efek dengan cleanup
- SideEffect untuk menyinkronkan ke external state
- Efek lifecycle awareness
- Resource cleanup dalam effects
- Lifecycle awareness dalam Compose
- Best practices untuk side effects

4 hari

## Prioritas
⭐⭐⭐ (Lifecycle awareness)

## Catatan
Side effects penting untuk menghubungkan Compose dengan dunia eksternal secara aman dari segi lifecycle.
