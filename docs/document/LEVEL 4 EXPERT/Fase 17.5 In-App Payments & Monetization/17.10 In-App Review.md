# In-App Review

## Deskripsi
In-App Review API memungkinkan permintaan ulasan dari pengguna tanpa meninggalkan aplikasi. Anda akan belajar implementasi permintaan ulasan yang tidak mengganggu.

## Tujuan Pembelajaran
- Menggunakan In-App Review API
- Mengimplementasikan permintaan ulasan yang tidak mengganggu
- Menggantikan pendekatan manual review prompts

## Materi
- In-App Review API implementation
- Review request timing
- User feedback collection
- Manual review prompts problems
- User feedback dalam aplikasi
- Review prompting best practices
- Rating collection dalam aplikasi

## Waktu
2 hari

## Prioritas
⭐⭐ (User feedback)

## Catatan
In-App Review API memberikan cara yang tidak mengganggu untuk mendapatkan ulasan dari pengguna.
