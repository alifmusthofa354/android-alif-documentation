# Hilt Setup

## Deskripsi
Hilt adalah dependency injection framework resmi dari Google untuk Android. Anda akan belajar bagaimana mengatur dan menggunakan Hilt dalam proyek Android.

## Tujuan Pembelajaran
- Mengatur Hilt dalam proyek Android
- Menggunakan @HiltAndroidApp dan @AndroidEntryPoint
- Menggantikan Dagger 2 manual setup

## Materi
- Setup Hilt dalam proyek
- @HiltAndroidApp annotation
- @AndroidEntryPoint untuk Activity/Fragment
- Integration dengan ViewModels
- Pengganti Dagger 2 manual
- Google recommended approach
- Dependency injection automation
