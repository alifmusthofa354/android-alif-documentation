# Gradle Configuration

## Deskripsi
Konfigurasi Gradle yang baik penting untuk maintainability. Anda akan belajar convention plugins, buildSrc, dan version catalogs untuk manajemen build yang lebih baik.

## Tujuan Pembelajaran
- Menggunakan convention plugins
- Mengelola buildSrc dan version catalogs
- Menggantikan konfigurasi duplikat

## Materi
- Convention plugins dalam Gradle
- buildSrc untuk shared build logic
- Version catalogs untuk dependency management
- Duplicate configs problems
- Maintainability dalam build configuration
- Gradle configuration best practices
- Build system optimization dalam modular apps
