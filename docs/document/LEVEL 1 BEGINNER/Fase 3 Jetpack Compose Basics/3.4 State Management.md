# State Management

## Deskripsi
State management adalah konsep penting dalam Jetpack Compose. Anda akan belajar bagaimana mengelola state di dalam komponen Compose.

## Tujuan Pembelajaran
- Memahami konsep state dalam Compose
- Menggunakan remember dan mutableStateOf
- Menggunakan rememberSaveable untuk menyimpan state

## Materi
- Konsep state dalam Compose
- remember dan mutableStateOf
- rememberSaveable
- State hoisting dasar
- Composition dan recomposition
- State vs instance variable

## Catatan
