# ANR Prevention

## Deskripsi
ANR (Application Not Responding) adalah masalah serius yang mempengaruhi pengalaman pengguna. Anda akan belajar teknik untuk mencegah dan men-debug ANR.

## Tujuan Pembelajaran
- Memahami penyebab ANR
- Menggunakan StrictMode untuk deteksi ANR
- Mendeteksi main thread operation yang tidak seharusnya

## Materi
- Apa itu ANR dan penyebabnya
- StrictMode setup dan penggunaan
- Deteksi operasi yang seharusnya tidak di main thread
- ANR troubleshooting techniques
- Main thread detection tools
- Penanganan tugas berat di background thread
- App responsiveness improvement

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (App responsiveness)

## Catatan
ANR prevention sangat penting untuk menjaga aplikasi tetap responsif dan memberikan pengalaman pengguna yang baik.
