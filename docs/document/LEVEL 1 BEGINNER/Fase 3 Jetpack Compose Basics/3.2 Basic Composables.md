# Basic Composables

## Deskripsi
Anda akan belajar tentang Composable functions dasar yang digunakan untuk membuat UI dalam Jetpack Compose, termasuk text, image, button, dan layout elements.

## Tujuan Pembelajaran
- Menggunakan Composable dasar seperti Text, Image, Button
- Membuat layout sederhana dengan Column, Row, Box
- Memahami perbedaan dengan View-based UI

## Materi
- Text composable
- Image composable
- Button dan variants-nya
- Column (sebagai pengganti LinearLayout vertikal)
- Row (sebagai pengganti LinearLayout horizontal)
- Box (sebagai pengganti FrameLayout)
- Perbedaan dengan TextView, ImageView, dll.

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (Building blocks)

## Catatan
Ini adalah building blocks utama dalam Jetpack Compose yang akan sering Anda gunakan.
