

## **3.8 Compose Previews** (2 hari) ⭐⭐

### **Teknologi Modern:**
- @Preview annotation
- Preview parameters
- Interactive previews

### **Teknologi yang Ditinggalkan:**
- Harus run app untuk lihat UI

### **Sub-topik Detail:**

#### 1. Basic @Preview
- @Preview annotation syntax
- Preview di Android Studio
- Refresh preview

#### 2. Preview Parameters
- name: memberikan nama preview
- showBackground: tampilkan background
- backgroundColor: warna background custom
- showSystemUi: tampilkan status bar & nav bar
- device: specify device (Devices.PIXEL_4, dll)
- uiMode: UI_MODE_NIGHT_YES untuk dark theme
- fontScale: test dengan font scale berbeda
- locale: test dengan bahasa berbeda
- widthDp, heightDp: custom size

#### 3. Multiple Previews
- Buat multiple @Preview untuk different states
- Light dan dark theme previews
- Different device sizes
- Different orientations

#### 4. @PreviewParameter
- Generate multiple previews dengan data berbeda
- PreviewParameterProvider implementation
- values() untuk provide sample data

#### 5. Preview Groups
- @Preview annotation pada annotation class
- Reusable preview configurations

#### 6. Interactive Preview Mode
- Deploy ke emulator dari preview
- Test interaksi langsung

#### 7. Preview Best Practices
- Preview semua states (empty, loading, error, success)
- Preview edge cases
- Keep preview functions lightweight
- Separate preview data dari production
