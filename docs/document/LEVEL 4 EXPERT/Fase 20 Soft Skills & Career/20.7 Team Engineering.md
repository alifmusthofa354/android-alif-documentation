Team engineering mencakup praktik untuk bekerja dalam tim besar. Anda akan belajar manajemen technical debt, feature toggles, dan distributed workflows.

## Tujuan Pembelajaran
- Mengelola technical debt
- Menggunakan feature toggles
- Mengelola distributed git workflows
- Memahami on-call mobile engineering

## Materi
- Technical debt management dalam tim
- Feature toggles implementation
- Distributed git workflows
- On-call mobile engineering practices
- Large team practices dalam development
- Engineering collaboration dalam tim besar
- Team scalability dalam engineering
