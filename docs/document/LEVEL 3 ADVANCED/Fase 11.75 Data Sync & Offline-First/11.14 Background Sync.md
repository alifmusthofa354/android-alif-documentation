# Background Sync

## Deskripsi
Background sync digunakan untuk sinkronisasi data secara otomatis. Anda akan belajar implementasi sync periodik dengan WorkManager dan constraint management.

## Tujuan Pembelajaran
- Mengimplementasikan background sync periodik
- Menggunakan WorkManager untuk scheduled sync
- Mengelola constraints dalam sync process

## Materi
- Background sync concepts
- Periodic sync dengan WorkManager
- Sync constraints (network, charging, time)
- Sync schedule management
- Data consistency dalam background
- Unreliable sync problems
- Data consistency melalui sync
- Scheduled sync dengan constraints

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Data consistency)

## Catatan
Background sync penting untuk menjaga konsistensi data antara lokal dan server secara otomatis.
