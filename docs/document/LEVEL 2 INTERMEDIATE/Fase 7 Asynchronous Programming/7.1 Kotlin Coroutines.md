# Kotlin Coroutines

## Deskripsi
Coroutines adalah pendekatan modern untuk pemrograman asynchronous di Kotlin. Anda akan belajar bagaimana menggunakan coroutines untuk operasi yang tidak memblokir UI.

## Tujuan Pembelajaran
- Memahami konsep Kotlin Coroutines
- Menggunakan suspend functions
- Mengelola CoroutineScope

## Materi
- Konsep Coroutines
- suspend functions
- CoroutineScope dan CoroutineContext
- launch vs async
- Pengganti AsyncTask, Thread, dan Handler
- Modern approach untuk async programming
- Structured concurrency
