# Canvas & Graphics

## Deskripsi
Canvas dalam Compose memungkinkan Anda menggambar grafik kustom secara langsung. Anda akan belajar DrawScope, Path, dan penggunaan shader dalam Compose.

## Tujuan Pembelajaran
- Menggunakan Canvas dalam Compose
- Menggambar dengan DrawScope
- Menggunakan Path dan Shaders (Skia backend)

## Materi
- Canvas dalam Compose
- DrawScope dan drawing operations
- Path untuk bentuk grafis kompleks
- Shader (Skia backend) dalam Compose
- DrawBehind dan DrawWithContent
- Pengganti Custom View onDraw
- Custom graphics implementation dalam Compose

## Waktu
4 hari

## Prioritas
⭐⭐ (Custom graphics)

## Catatan
Canvas dalam Compose memberikan kekuatan untuk menggambar grafis kustom secara efisien.
