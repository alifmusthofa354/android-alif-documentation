# Repository Pattern

## Deskripsi
Repository pattern digunakan untuk memisahkan logika business dari source data. Anda akan belajar bagaimana mengimplementasikan repository layer untuk manajemen data yang bersih.

## Tujuan Pembelajaran
- Memahami konsep Repository pattern
- Memisahkan data source dari business logic
- Menggunakan repository dalam arsitektur clean

## Materi
- Konsep Repository pattern
- Implementasi Repository abstract class/interface
- Data source separation
- Repository dalam konteks MVVM
- Penghindaran direct API calls in UI
- Single source of truth

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Clean architecture)

## Catatan
Repository pattern adalah bagian penting dari clean architecture yang memisahkan sumber data dari logika aplikasi.
