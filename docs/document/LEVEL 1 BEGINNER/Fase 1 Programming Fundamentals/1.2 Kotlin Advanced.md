# Kotlin Advanced

## Deskripsi
Setelah menguasai dasar-dasar Kotlin, pada topik ini Anda akan belajar fitur-fitur lanjutan yang membuat Kotlin sangat powerful dan aman untuk pengembangan aplikasi.

## Tujuan Pembelajaran
- Memahami konsep null safety dan pentingnya untuk mencegah NullPointerException
- Menggunakan extension functions untuk memperluas kelas bawaan
- Menggunakan higher-order functions dan lambda untuk pemrograman fungsional

## Materi
- Null safety (null, non-null, safe call operator, elvis operator)
- Extension functions
- Higher-order functions
- Lambda expressions
- Function literals with receiver
- Inline functions (optional)

## Waktu
1 minggu

## Prioritas
⭐⭐⭐ (Foundation penting)

## Catatan
Fitur-fitur ini adalah bagian penting dari Kotlin yang membuat pengembangan lebih aman dan produktif.
