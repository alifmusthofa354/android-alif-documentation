# Sensors & Location

## Deskripsi
Sensor dan Location API memungkinkan pembuatan aplikasi yang context-aware. Anda akan belajar menggunakan Location API dan berbagai sensor dalam perangkat Android.

## Tujuan Pembelajaran
- Menggunakan Location API untuk pelacakan lokasi
- Mengakses sensor seperti accelerometer dan gyroscope
- Menggantikan pendekatan location API yang deprecated

## Materi
- Location API implementation
- GPS, Network, dan Fused Location Provider
- Accelerometer, gyroscope, dan sensor lainnya
- Sensor Manager dalam Android
- Deprecated location APIs problems
- Context-aware apps dalam Android
- Location dan sensor data processing

## Waktu
4 hari

## Prioritas
⭐⭐ (Context-aware apps)

## Catatan
Sensors dan Location API penting untuk aplikasi yang memerlukan konteks lingkungan pengguna.
