# MVI Concepts

## Deskripsi
MVI (Model-View-Intent) adalah arsitektur yang menggunakan unidirectional data flow. Anda akan belajar konsep dasar MVI dan perbedaannya dari arsitektur lain.

## Tujuan Pembelajaran
- Memahami konsep MVI (Model-View-Intent)
- Mengelola unidirectional data flow
- Memahami perbedaan dari arsitektur lain

## Materi
- Konsep MVI (Model-View-Intent)
- Unidirectional data flow
- Intent sebagai input dari user
- State sebagai output ke UI
- Perbedaan dengan MVVM/MVP
- Modern architecture pattern
- Data flow dalam MVI
- State management dalam konteks MVI

4 hari

## Prioritas
⭐⭐ (Modern architecture)

## Catatan
MVI adalah pendekatan modern untuk arsitektur yang memberikan prediktabilitas state.
