# Remote Config

## Deskripsi
Firebase Remote Config memungkinkan konfigurasi aplikasi yang bisa diubah tanpa update aplikasi. Anda akan belajar implementasi feature flags dan A/B testing.

## Tujuan Pembelajaran
- Menggunakan Firebase Remote Config
- Mengimplementasikan feature flags
- Melakukan A/B testing

## Materi
- Remote Config setup dan usage
- Feature flags implementation
- A/B testing dengan Remote Config
- Dynamic configuration dalam aplikasi
- Hardcoded configs problems
- Remote configuration management
- Configuration updates tanpa deployment

3 hari

## Prioritas
⭐⭐ (Feature flags)

## Catatan
Remote Config penting untuk pengujian fitur dan konfigurasi dinamis tanpa harus mengupdate aplikasi.
