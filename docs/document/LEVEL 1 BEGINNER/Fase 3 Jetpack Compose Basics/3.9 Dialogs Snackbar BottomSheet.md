# Dialogs, Snackbar, BottomSheet

## Deskripsi
Material 3 Composables menyediakan komponen standar untuk komunikasi dengan pengguna seperti AlertDialog, Snackbar, dan BottomSheet. Anda akan belajar cara menggunakannya dalam Compose.

## Tujuan Pembelajaran
- Menggunakan AlertDialog dalam Compose
- Menampilkan Snackbar untuk pesan singkat
- Menggunakan BottomSheet untuk konten tambahan

## Materi
- AlertDialog dalam Compose
- Basic dan Action Snackbar
- ModalBottomSheet dan StandardBottomSheet
- State management untuk komponen-komponen ini
- Custom dialogs vs Material components
- Best practices penggunaan

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Standard communication)

## Catatan
Komponen-komponen ini menyediakan cara standar dan familiar bagi pengguna untuk berinteraksi.
