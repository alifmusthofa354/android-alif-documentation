

## **3.3 Layouts & Modifiers** (4 hari) ⭐⭐⭐

### **Teknologi Modern:**
- LazyColumn, LazyRow
- Spacer
- Modifier (comprehensive)

### **Teknologi yang Ditinggalkan:**
- RecyclerView
- ConstraintLayout

### **Sub-topik Detail:**

#### 1. LazyColumn
- Basic LazyColumn dengan items()
- items() dengan list data
- itemsIndexed() dengan index
- item() untuk single item
- Key parameter untuk stabilitas
- contentPadding
- reverseLayout
- State management dengan rememberLazyListState()
- Scroll to position programmatically
- Sticky headers dengan stickyHeader()

#### 2. LazyRow
- Horizontal scrolling list
- Similar APIs dengan LazyColumn
- Use cases: horizontal carousels, chips

#### 3. LazyVerticalGrid
- Grid dengan columns fixed atau adaptive
- GridCells.Fixed(count)
- GridCells.Adaptive(minSize)
- Span untuk item berbeda

#### 4. Spacer
- Memberikan spacing fixed
- Modifier.height() dan width()
- Alternatif padding

#### 5. Modifier Deep Dive
**Size Modifiers:**
- size(width, height)
- width(), height()
- fillMaxWidth(), fillMaxHeight(), fillMaxSize()
- wrapContentSize(), wrapContentWidth(), wrapContentHeight()
- requiredSize() vs size()
- defaultMinSize()

**Padding & Margin:**
- padding(all)
- padding(horizontal, vertical)
- padding(start, top, end, bottom)
- Tidak ada margin di Compose (gunakan padding parent)

**Background & Border:**
- background(color)
- background(brush) untuk gradient
- border(width, color, shape)

**Shape & Clipping:**
- clip(shape) - RoundedCornerShape, CircleShape
- Custom shapes

**Click & Interaction:**
- clickable { }
- clickable dengan indication dan interactionSource
- combinedClickable untuk long press

**Alignment & Position:**
- align() dalam Box/Column/Row
- offset(x, y)
- absoluteOffset()

**Other Important Modifiers:**
- alpha() untuk transparency
- rotate(), scale()
- shadow() untuk elevation effect
- weight() untuk flexible sizing dalam Row/Column
- aspectRatio()

#### 6. Modifier Order
- Pentingnya urutan modifier
- Contoh: padding -> background vs background -> padding

#### 7. Custom Modifiers
- Membuat extension function Modifier
- Reusable modifier chains
