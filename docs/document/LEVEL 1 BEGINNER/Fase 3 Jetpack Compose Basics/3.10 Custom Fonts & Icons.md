# Custom Fonts & Icons

## Deskripsi
Untuk branding dan tampilan yang menarik, aplikasi sering menggunakan font dan ikon kustom. Anda akan belajar bagaimana mengimplementasikannya dalam Jetpack Compose.

## Tujuan Pembelajaran
- Menggunakan custom fonts dalam Compose
- Menggunakan Icon composables
- Mengatur typography konsisten

## Materi
- Menggunakan custom fonts
- Icon composable dan Material Icons
- Typography dalam Compose
- Font specification dan loading
- Downloadable fonts
- Branding dan detail UI

1 hari

## Catatan
Custom fonts dan icons penting untuk branding dan tampilan aplikasi yang unik.
