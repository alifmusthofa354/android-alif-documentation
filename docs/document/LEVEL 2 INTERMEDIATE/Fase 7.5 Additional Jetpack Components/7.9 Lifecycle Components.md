# Lifecycle Components

## Deskripsi
Lifecycle components membantu Anda membuat komponen yang aware terhadap lifecycle tanpa tight coupling. Anda akan belajar lifecycleScope dan repeatOnLifecycle.

## Tujuan Pembelajaran
- Menggunakan lifecycleScope untuk korutin yang otomatis dikelola
- Menggunakan repeatOnLifecycle untuk mengelola lifecycle dengan benar
- Menghindari manual lifecycle handling

## Materi
- lifecycleScope
- repeatOnLifecycle
- Lifecycle-aware components
- Penghindaran memory leaks terkait lifecycle
- Proper coroutine management dalam lifecycle
- Integration dengan Fragment dan Activity
- Lifecycle-aware approach

## Catatan
