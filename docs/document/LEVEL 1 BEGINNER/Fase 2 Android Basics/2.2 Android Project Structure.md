# Android Project Structure

## Deskripsi
Memahami struktur project Android sangat penting untuk pengembangan yang efisien. Anda akan belajar tentang folder-folder penting dan konfigurasi dasar project.

## Tujuan Pembelajaran
- Memahami struktur folder project Android
- Mengenal file-file konfigurasi penting
- Memahami dasar Gradle dalam project Android

## Materi
- Struktur folder (app, src, main, java/kotlin, res, assets)
- File AndroidManifest.xml
- build.gradle (project dan app level)
- Gradle Kotlin DSL
- Folder res (drawable, layout, values)
- File-file penting lainnya (proguard-rules.pro, gradle.properties)

## Waktu
2 hari

## Prioritas
⭐⭐ (Memahami struktur project)

## Catatan
Pemahaman tentang struktur project akan membantu Anda dalam navigasi dan organisasi kode.
