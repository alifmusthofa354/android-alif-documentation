## 1Ô∏è‚É£ Pengenalan Material Design 3 üé®

### Konsep Dasar
Material Design 3 (M3) seperti lukisan hidup üñºÔ∏è - bisa berubah warna dan bentuk berdasarkan preferensi pengguna dan lingkungan!

### Material You Concept
Material You adalah fitur Android 12+ yang memungkinkan sistem warna dinamis berdasarkan wallpaper pengguna.

### Perbedaan M2 vs M3
| Material Design 2 | Material Design 3 |
|------------------|-------------------|
| Warna statis | Warna dinamis |
| Tidak ada surface tints | Menggunakan surface tints |
| Lebih datar | Lebih hidup dan dinamis |
| Kustomisasi terbatas | Kustomisasi lebih luas |

fun MyComposeAppTheme(
    darkTheme: Boolean = isSystemInDarkTheme(),
    // Dynamic color is available on Android 12+
    dynamicColor: Boolean = true,
    content: @Composable () -> Unit
) {
    val colorScheme = when {
        dynamicColor && android.os.Build.VERSION.SDK_INT >= android.os.Build.VERSION_CODES.S -> {
            val context = androidx.compose.ui.platform.LocalContext.current
            if (darkTheme) dynamicDarkColorScheme(context) else dynamicLightColorScheme(context)
        }
        darkTheme -> DarkColorScheme
        else -> LightColorScheme
    }

    // Update system bars based on theme
    val systemUiController = rememberSystemUiController()
    SideEffect {
        systemUiController.setSystemBarsColor(
            color = colorScheme.primary,
            darkIcons = !darkTheme
        )
    }

    MaterialTheme(
        colorScheme = colorScheme,
        typography = Typography,
        shapes = Shapes,
        content = content
    )
}

### üéì Penjelasan Konsep:
**Dark Theme seperti Mode Malam:**
- Mengurangi kelelahan mata di lingkungan gelap
- Menghemat baterai di layar OLED
- Memberikan pengalaman visual yang berbeda

**Implementasi Dark Theme:**
- `isSystemInDarkTheme()`: Mendeteksi mode sistem
- `darkColorScheme`: Skema warna untuk tema gelap
- `MaterialTheme`: Menggunakan skema warna sesuai mode

**Dynamic Theme Switching:**
- Mengganti tema dalam runtime
- Menggunakan state untuk menyimpan preferensi
- Komponen merekomposisi saat tema berubah
