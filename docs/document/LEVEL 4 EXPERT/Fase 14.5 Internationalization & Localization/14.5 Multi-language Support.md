# Multi-language Support

## Deskripsi
Dukungan multi-bahasa penting untuk aplikasi global. Anda akan belajar implementasi string lokal per locale dan plural dalam aplikasi Android.

## Tujuan Pembelajaran
- Menggunakan strings.xml per locale
- Mengelola plural dalam berbagai bahasa
- Menggantikan hardcoded strings

## Materi
- Strings.xml untuk berbagai locale
- Plurals untuk berbagai bahasa
- String resource management
- Hardcoded strings problems
- Global apps requirement
- Internationalization process
- Locale-specific string resources

## Catatan
Multi-language support penting untuk menjangkau pasar global dan memenuhi kebutuhan pengguna lokal.
