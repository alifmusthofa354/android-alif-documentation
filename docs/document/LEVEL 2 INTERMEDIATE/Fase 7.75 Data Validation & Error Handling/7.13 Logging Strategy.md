# Logging Strategy

## Deskripsi
Strategi logging yang baik sangat penting untuk debugging dan monitoring aplikasi. Anda akan belajar menggunakan Timber dan Kermit untuk structured logging.

## Tujuan Pembelajaran
- Menggunakan Timber untuk logging
- Menggunakan Kermit untuk structured logging
- Menerapkan logging strategy yang efisien

## Materi
- Setup Timber
- Structured logging dengan Kermit
- Log levels dan penggunaannya
- Logging dalam environment yang berbeda
- Menghindari println everywhere
- Debugging efficiency melalui logging
- Production logging practices

2 hari

## Prioritas
⭐⭐⭐ (Debugging efficiency)

## Catatan
Logging strategy yang baik sangat membantu dalam debugging dan monitoring aplikasi production.
