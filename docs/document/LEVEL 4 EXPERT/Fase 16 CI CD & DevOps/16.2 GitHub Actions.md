# GitHub Actions

## Deskripsi
GitHub Actions adalah platform CI/CD modern untuk otomatisasi workflow. Anda akan belajar implementasi automated workflows dalam pengembangan Android.

## Tujuan Pembelajaran
- Menggunakan GitHub Actions untuk CI/CD
- Mengimplementasikan automated workflows
- Menggantikan pendekatan Jenkins

## Materi
- GitHub Actions setup dan configuration
- Workflow definition dalam YAML
- CI/CD untuk Android projects
- Automated testing dan building
- Jenkins problems
- Modern CI/CD dalam Android development
- Automated workflows dalam repository
