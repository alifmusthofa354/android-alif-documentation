# Room Relations

## Deskripsi
Dalam aplikasi nyata, data sering memiliki relasi antar tabel. Anda akan belajar bagaimana mengimplementasikan relasi dalam Room database.

## Tujuan Pembelajaran
- Mengimplementasikan relasi one-to-many dalam Room
- Mengimplementasikan relasi many-to-many dalam Room
- Menggantikan multiple tables tanpa relation

## Materi
- Relasi one-to-many dalam Room (@Embedded, @Relation)
- Relasi many-to-many dengan junction table
- Foreign key constraints
- Query kompleks dengan relasi
- Best practices dalam implementasi relasi
- Complex data relationships management

## Waktu
3 hari

## Prioritas
⭐⭐ (Complex data)

## Catatan
Relasi dalam Room memungkinkan struktur data yang kompleks dan konsisten.
