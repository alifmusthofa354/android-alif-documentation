# OOP Concepts

## Deskripsi
Object-Oriented Programming (OOP) adalah paradigma penting dalam pengembangan software. Anda akan belajar konsep-konsep OOP dan bagaimana menerapkannya di Kotlin.

## Tujuan Pembelajaran
- Memahami konsep dasar OOP
- Mampu membuat kelas dan objek
- Memahami inheritance, interface, dan abstraction

## Materi
- Classes dan Objects
- Inheritance
- Interfaces
- Abstract class
- Properties dan Fields
- Data classes
- Companion objects
