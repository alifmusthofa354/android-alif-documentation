# Screenshot Testing

## Deskripsi
Screenshot testing digunakan untuk deteksi visual regression dalam aplikasi. Anda akan belajar Paparazzi dan Roborazzi untuk testing visual.

## Tujuan Pembelajaran
- Menggunakan Paparazzi untuk screenshot testing
- Menggunakan Roborazzi untuk visual testing
- Menggantikan pendekatan manual visual testing

## Materi
- Paparazzi setup dan usage
- Roborazzi untuk screenshot dalam Compose
- Visual regression detection
- Testing UI dengan pendekatan visual
- Pengganti manual visual testing
- Screenshot testing dalam CI
- Visual consistency checking

## Waktu
3 hari

## Prioritas
⭐⭐ (Visual regression)

## Catatan
Screenshot testing penting untuk mendeteksi perubahan visual yang tidak diinginkan.
