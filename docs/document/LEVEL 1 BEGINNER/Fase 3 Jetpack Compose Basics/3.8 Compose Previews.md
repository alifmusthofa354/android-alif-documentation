# Compose Previews

## Deskripsi
Compose Preview memungkinkan Anda melihat tampilan komponen langsung di Android Studio tanpa harus menjalankan aplikasi. Ini mempercepat iterasi UI.

## Tujuan Pembelajaran
- Menggunakan @Preview annotation
- Membuat preview dengan parameter berbeda
- Mempercepat iterasi UI

## Materi
- @Preview annotation
- PreviewParameter
- Multiple previews
- Device dan size variations
- Dark mode preview
- Pengganti dari run app untuk melihat UI
- Interaction dengan UI dalam preview

## Waktu
2 hari

## Prioritas
⭐⭐ (Fast iteration)

## Catatan
Preview sangat meningkatkan produktivitas dalam pengembangan UI.
