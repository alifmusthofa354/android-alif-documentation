# Collections & Generics

## Deskripsi
Kotlin menyediakan berbagai jenis collections yang powerful dan mudah digunakan. Anda akan belajar tentang Kotlin Collections dan bagaimana menggunakannya secara efektif.

## Tujuan Pembelajaran
- Memahami berbagai jenis collections di Kotlin
- Mampu menggunakan collections untuk manipulasi data
- Memahami konsep sequences untuk efisiensi

## Materi
- List (Mutable dan Immutable)
- Set (Mutable dan Immutable)
- Map (Mutable dan Immutable)
- Operasi pada collections (filter, map, forEach, etc.)
- Sequences vs Collections
- Generics di Kotlin
