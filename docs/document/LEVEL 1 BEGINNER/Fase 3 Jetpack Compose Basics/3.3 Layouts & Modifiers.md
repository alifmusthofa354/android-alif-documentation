# Layouts & Modifiers

## Deskripsi
Modifiers dan layout composables adalah inti dari pembuatan UI dalam Jetpack Compose. Anda akan belajar cara menyusun komponen dan menggabungkan berbagai modifikasi.

## Tujuan Pembelajaran
- Menggunakan Modifiers untuk mengubah tampilan dan perilaku composables
- Membuat list efisien dengan Lazy composables
- Menggunakan Spacer untuk padding dan spacing

## Materi
- Konsep Modifiers
- Penggunaan modifier dalam urutan yang benar
- LazyColumn dan LazyRow (sebagai pengganti RecyclerView)
- Spacer untuk spacing
- Nested layouts
- Konsep parenting dalam Compose

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (UI composition)

## Catatan
Modifiers adalah cara utama dalam Compose untuk menggabungkan styling, layout, dan event handling.
