# KSP (Kotlin Symbol Processing)

## Deskripsi
Kotlin Symbol Processing (KSP) adalah framework untuk code generation yang lebih cepat dan efisien dibanding Annotation Processing. Anda akan belajar bagaimana menggunakannya untuk generasi kode.

## Tujuan Pembelajaran
- Memahami konsep KSP
- Menggunakan KSP untuk code generation
- Membandingkan dengan Annotation Processing

## Materi
- Konsep Kotlin Symbol Processing
- Perbedaan KSP dan Annotation Processing
- Setup KSP dalam proyek
- Performance benefits dari KSP
- KSP for code generation
- Modern code generation approach
- Integration dengan Hilt dan library lain

3 hari

## Prioritas
⭐⭐ (Modern code generation)

## Catatan
KSP adalah pendekatan modern untuk code generation yang lebih cepat dari Annotation Processing.
