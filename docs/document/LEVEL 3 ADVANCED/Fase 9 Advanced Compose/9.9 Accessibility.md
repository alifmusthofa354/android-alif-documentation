# Accessibility

## Deskripsi
Accessibility penting untuk membuat aplikasi bisa diakses oleh semua pengguna termasuk yang memiliki disabilitas. Anda akan belajar bagaimana membuat aplikasi Compose yang inklusif.

## Tujuan Pembelajaran
- Menggunakan Semantics dalam Compose
- Menyediakan contentDescription untuk screen readers
- Mengelola accessibility actions

## Materi
- Konsep accessibility dalam aplikasi
- Semantics dalam Compose
- ContentDescription untuk gambar dan ikon
- Accessibility actions
- Testing accessibility
- Ignoring accessibility problems
- Inclusive design dalam Compose
- Accessibility services integration

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Inclusive design)

## Catatan
Accessibility adalah bagian penting dari aplikasi yang ingin mencakup semua pengguna.
