# Mocking

## Deskripsi
Mocking digunakan untuk menggantikan dependensi nyata dalam testing. Anda akan belajar MockK yang dirancang khusus untuk Kotlin dengan pendekatan yang Kotlin-friendly.

## Tujuan Pembelajaran
- Menggunakan MockK untuk mocking
- Melakukan mockk, every, dan verify
- Menggantikan Mockito untuk Kotlin

## Materi
- Setup MockK
- mockk untuk membuat mock object
- every untuk mendefinisikan behavior
- verify untuk memastikan interaksi
- Keunggulan MockK dari Mockito
- Kotlin-friendly mocking framework
- Mocking dalam context Kotlin

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (Kotlin-friendly)

## Catatan
MockK adalah pilihan terbaik untuk Kotlin karena fitur-fitur yang dirancang khusus untuk bahasa ini.
