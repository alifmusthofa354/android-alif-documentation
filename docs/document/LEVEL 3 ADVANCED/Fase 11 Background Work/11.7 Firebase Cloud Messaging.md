# Firebase Cloud Messaging

## Deskripsi
Firebase Cloud Messaging (FCM) adalah layanan push notification dari Google. Anda akan belajar implementasi FCM dan push notification pipeline dalam aplikasi.

## Tujuan Pembelajaran
- Mengimplementasikan Firebase Cloud Messaging
- Menangani push notifications
- Mengelola notification pipeline

## Materi
- Setup FCM dalam aplikasi
- FCM message handling
- Notification pipeline dari server ke device
- Topic messaging dan device token
- Pengganti GCM (deprecated)
- Push messaging dalam aplikasi
- FCM service integration

## Catatan
