# Animations

## Deskripsi
Animasi membuat aplikasi lebih hidup dan memberikan feedback kepada pengguna. Anda akan belajar berbagai teknik animasi dalam Jetpack Compose.

## Tujuan Pembelajaran
- Menggunakan AnimatedVisibility untuk animasi muncul/hilang
- Menggunakan animateContentSize untuk animasi ukuran
- Menggunakan Transition untuk animasi kompleks

## Materi
- AnimatedVisibility
- animateContentSize
- Transition API
- AnimateFloatAsState
- Animasi gesture-driven
- Pengganti View animations
- Smooth UX melalui animasi
- Animation specs dan customization

## Waktu
4 hari

## Prioritas
⭐⭐ (Smooth UX)

## Catatan
Animasi yang baik meningkatkan pengalaman pengguna dan memberikan feedback visual.
