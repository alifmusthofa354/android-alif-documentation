# Image Loading

## Deskripsi
Coil adalah library image loading yang dirancang khusus untuk Kotlin dan Compose. Anda akan belajar bagaimana mengimplementasikan loading gambar yang efisien.

## Tujuan Pembelajaran
- Menggunakan Coil untuk loading gambar
- Mengimplementasikan caching gambar
- Mengoptimalkan penggunaan memory untuk gambar

## Materi
- Konfigurasi Coil
- Loading gambar dalam Compose
- Caching strategies
- Transformation dan filtering
- Placeholder dan error images
- Memory management untuk gambar
- Pengganti Picasso dan Glide untuk project baru

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Efficient image handling)

## Catatan
Coil adalah pilihan terbaik untuk project baru karena optimalisasi khusus untuk Kotlin dan Compose.
