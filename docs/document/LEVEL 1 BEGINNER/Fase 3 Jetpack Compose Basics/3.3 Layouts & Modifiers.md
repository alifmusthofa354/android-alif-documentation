## 1Ô∏è‚É£ LazyColumn: Tampilan List Vertikal üß©

### Konsep Dasar
LazyColumn seperti tangga ü™ú yang panjang - hanya item yang terlihat yang dirender, membuat aplikasi tetap cepat!

### Perbedaan dengan Column:
- **Column**: Merender semua item sekaligus (tidak efisien untuk list panjang)
- **LazyColumn**: Merender hanya item yang terlihat (efisien dan cepat)

### Implementation
**ui/screens/LazyColumnExamplesScreen.kt**
```kotlin
package com.example.mycomposeapp.ui.screens

import androidx.compose.foundation.layout.*
import androidx.compose.foundation.lazy.LazyColumn
import androidx.compose.foundation.lazy.items
import androidx.compose.foundation.lazy.itemsIndexed
import androidx.compose.foundation.lazy.rememberLazyListState
import androidx.compose.foundation.rememberScrollState

@Composable

    val scrollState = rememberLazyListState()
    
    Column(
        modifier = Modifier
            .fillMaxSize()

            modifier = Modifier
                .fillMaxWidth()
                .height(200.dp)
                .background(Color(0xFFE3F2FD)),
            state = scrollState
        ) {
            items((1..10).toList()) { index ->
                Text(
                    text = "üì¶ Item $index",
                    modifier = Modifier
                        .fillMaxWidth()
                        .padding(8.dp)
                        .background(Color(0xFF2196F3))
                        .padding(16.dp),
                    color = Color.White
                )
            }
        }

        // 2. LazyColumn dengan data real
        Text("2Ô∏è‚É£ LazyColumn dengan Data", fontSize = 20.sp, fontWeight = FontWeight.Bold)
        
        LazyColumn(
            modifier = Modifier
                .fillMaxWidth()

            verticalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            val items = listOf(
                "üì± Ponsel Canggih",
                "üíª Laptop Gaming",
                "üéß Headphone Wireless",
                "‚åö Smart Watch",
                "üì∑ Kamera DSLR"

                    modifier = Modifier
                        .fillMaxWidth()
                        .padding(4.dp)

            modifier = Modifier

                    verticalAlignment = Alignment.CenterVertically,
                    horizontalArrangement = Arrangement.spacedBy(8.dp)

                        text = "${index + 1}",
                        color = Color.White,
                        fontWeight = FontWeight.Bold

            modifier = Modifier
                .fillMaxWidth()
                .height(150.dp)
                .background(Color(0xFFFFEBEE)),

            verticalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            items((1..5).toList()) { index ->

                    Text(
                        text = "üéÅ Item $index dengan padding",

        Text("6Ô∏è‚É£ reverseLayout (urutan terbalik)", fontSize = 20.sp, fontWeight = FontWeight.Bold)
        
        LazyColumn(
            modifier = Modifier
                .fillMaxWidth()
                .height(150.dp)
                .background(Color(0xFFE1F5FE)),
            reverseLayout = true
        ) {
            items((1..5).toList()) { index ->
                Text(
                    text = "üîÑ Terbalik: Item $index",
                    modifier = Modifier
                        .fillMaxWidth()
                        .padding(8.dp)
                        .background(Color(0xFF039BE5))
                        .padding(12.dp),
                    color = Color.White
                )
            }
        }

        // 7. State management untuk scroll
        Text("7Ô∏è‚É£ State Management (Scroll)", fontSize = 20.sp, fontWeight = FontWeight.Bold)
        
        Row(
            modifier = Modifier.fillMaxWidth(),
            horizontalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            Button(
                onClick = { scrollState.animateScrollToItem(0) },
                modifier = Modifier.weight(1f)
            ) {
                Text("Top")
            }
            Button(
                onClick = { scrollState.animateScrollToItem(5) },
                modifier = Modifier.weight(1f)
            ) {
                Text("Item 5")
            }
            Button(
                onClick = { scrollState.animateScrollToItem(9) },
                modifier = Modifier.weight(1f)
            ) {
                Text("Bottom")
            }
        }
    }
}
```

### üéì Penjelasan Konsep:
**LazyColumn seperti Tangga:**
- Hanya item yang terlihat di layar yang dirender
- Hemat memory dan performa
- Cocok untuk list panjang

**Fungsi penting:**
- `items(data)`: Untuk list biasa
- `itemsIndexed(data)`: Dapatkan index juga
- `item()`: Tambah item single (header/footer)
- `contentPadding`: Tambah padding dalam list
- `reverseLayout`: Urutan terbalik
- `rememberLazyListState()`: Manajemen state scroll
