

## **3.7 Lists & Grids** (3 hari) ⭐⭐⭐

### **Teknologi Modern:**
- LazyColumn
- LazyVerticalGrid
- items() DSL

### **Teknologi yang Ditinggalkan:**
- RecyclerView adapter
- ViewHolder pattern

### **Sub-topik Detail:**

#### 1. LazyColumn Advanced
- items() dengan list
- itemsIndexed() dengan index
- item() untuk single item atau mixed content
- Key parameter untuk item identity
- contentType untuk performance
- contentPadding untuk padding list
- reverseLayout untuk reverse order
- LazyListState untuk control scroll

#### 2. LazyRow Advanced
- Similar dengan LazyColumn
- Horizontal scrolling use cases
- Horizontal spacing dengan Arrangement

#### 3. LazyVerticalGrid
- GridCells.Fixed(columnCount)
- GridCells.Adaptive(minSize) untuk responsive
- items() dalam grid
- Span untuk item dengan width berbeda

#### 4. Item Keys
- Pentingnya stable keys
- Performance optimization
- Avoid using index as key

#### 5. Scroll State & Control
- rememberLazyListState()
- Scroll to position: animateScrollToItem()
- firstVisibleItemIndex
- Detect scroll: derivedStateOf
- Show/hide FAB on scroll

#### 6. Sticky Headers
- stickyHeader() dalam LazyColumn
- Use cases: grouped lists

#### 7. Item Animations
- Built-in animations
- Custom item animations dengan animateItemPlacement()

#### 8. Load More / Pagination
- Detect when reaching end of list
- Show loading indicator
- LaunchedEffect untuk load data

#### 9. Pull to Refresh
- Implementasi pull-to-refresh pattern
- SwipeRefresh (Accompanist atau manual)

#### 10. Empty State & Error Handling
- Show placeholder when list empty
- Error state handling
- Retry mechanism
