# Programming Logic

## Deskripsi
Algoritma dasar dan problem solving adalah foundation berpikir dalam pemrograman. Ini merupakan kemampuan dasar yang harus dimiliki sebelum mempelajari bahasa pemrograman.

## Tujuan Pembelajaran
- Memahami konsep algoritma
- Mampu menganalisis masalah dan membuat solusi langkah-demi-langkah
- Mengembangkan kemampuan problem solving

## Materi
- Pengertian algoritma dan pseudocode
- Struktur kontrol: percabangan (if-else) dan perulangan (loop)
- Teknik pemecahan masalah
- Flowchart dan diagram alur
- Contoh kasus sederhana

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Wajib dikuasai)

## Catatan
Foundation berpikir untuk semua konsep pemrograman yang akan dipelajari selanjutnya.
