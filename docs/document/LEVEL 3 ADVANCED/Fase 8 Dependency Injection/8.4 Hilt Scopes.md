# Hilt Scopes

## Deskripsi
Scopes dalam Hilt digunakan untuk mengontrol lifecycle dari dependensi. Anda akan belajar berbagai scope dan bagaimana menggunakannya untuk manajemen lifecycle.

## Tujuan Pembelajaran
- Menggunakan @Singleton, @ActivityScoped, @ViewModelScoped
- Memahami lifecycle management dalam DI
- Mengelola dependensi berdasarkan lifecycle

## Materi
- @Singleton scope
- @ActivityScoped
- @ViewModelScoped
- Lifecycle management dalam DI
- Dependency lifecycle matching
- Scope dalam konteks Android components
- Lifecycle management best practices

3 hari

## Prioritas
⭐⭐ (Lifecycle management)

## Catatan
Scopes memungkinkan pengelolaan lifecycle dependensi yang konsisten dengan komponen Android.
