# Setup Environment

## Deskripsi
Sebelum mulai membuat aplikasi Android, Anda perlu mengatur environment development. Di sini Anda akan belajar menginstal dan mengkonfigurasi Android Studio dan Gradle.

## Tujuan Pembelajaran
- Menginstal Android Studio terbaru
- Memahami Gradle Kotlin DSL
- Mengatur environment development

## Materi
- Download dan instal Android Studio (Hedgehog+)
- Pengenalan interface Android Studio
- Instalasi SDK
- Gradle Kotlin DSL vs Groovy DSL
- Konfigurasi gradle.properties
- Pembuatan project pertama

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Install & konfigurasi)

## Catatan
Ini adalah langkah awal penting untuk mulai pengembangan Android.
