# Room Database

## Deskripsi
Room adalah database SQLite abstraction dari Android Jetpack yang menyediakan database lokal yang kuat dan mudah digunakan. Anda akan belajar bagaimana mengimplementasikan Room dalam aplikasi Anda.

## Tujuan Pembelajaran
- Menggunakan Room sebagai abstraction database
- Membuat DAO, Entity, dan Database dalam Room
- Menggantikan raw SQLite usage

## Materi
- Konfigurasi Room
- Room @Entity untuk mendefinisikan tabel
- Room @Dao untuk mendefinisikan akses data
- RoomDatabase class
- Relations dalam Room
- Migration strategies
- Local persistence dengan Room
