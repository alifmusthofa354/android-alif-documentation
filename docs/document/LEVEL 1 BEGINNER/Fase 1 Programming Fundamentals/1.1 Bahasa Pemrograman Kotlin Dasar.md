# Bahasa Pemrograman (Kotlin Dasar)

## Deskripsi
Kotlin adalah bahasa utama untuk pengembangan Android modern. Pada fase ini, Anda akan belajar dasar-dasar Kotlin termasuk variabel, fungsi, kelas, dan objek.

## Tujuan Pembelajaran
- Memahami sintaks dasar Kotlin
- Mampu membuat variabel dan fungsi sederhana
- Memahami konsep OOP sederhana di Kotlin

## Materi
- Sintaks dasar Kotlin
- Variabel (val, var)
- Tipe data (String, Int, Double, Boolean, dll)
- Fungsi (function)
- Nullable dan non-nullable types
- Control flow (if/else, when)
- Loop (for, while)

## Waktu
2 minggu

## Prioritas
⭐⭐⭐ (Wajib, fokus syntax Kotlin)

## Catatan
Fokus utama adalah memahami sintaks Kotlin karena ini akan digunakan di seluruh proyek Android.
