# MVVM Architecture

## Deskripsi
MVVM (Model-View-ViewModel) adalah arsitektur yang direkomendasikan untuk aplikasi Android modern. Anda akan belajar bagaimana mengimplementasikan MVVM dengan ViewModel dan StateFlow.

## Tujuan Pembelajaran
- Memahami konsep arsitektur MVVM
- Menggunakan ViewModel untuk manajemen state
- Menggunakan StateFlow untuk manajemen state reaktif

## Materi
- Konsep MVVM (Model, View, ViewModel)
- Implementasi ViewModel
- StateFlow vs LiveData
- Penggunaan ViewModel dalam Compose
- Keunggulan MVVM dibanding MVP/MVC
- Praktik terbaik dalam implementasi

## Catatan
MVVM adalah standar industri untuk arsitektur aplikasi Android modern.
