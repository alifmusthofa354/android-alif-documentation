# CompositionLocal

## Deskripsi
CompositionLocal digunakan untuk menyediakan nilai ke bawah tree composables tanpa harus melewatkan melalui props. Anda akan belajar bagaimana menggunakannya sebagai alternatif untuk prop drilling.

## Tujuan Pembelajaran
- Menggunakan CompositionLocal untuk menyediakan nilai
- Menghindari prop drilling
- Memahami kapan menggunakan CompositionLocal

## Materi
- Konsep CompositionLocal
- Providing values down the tree
- CompositionLocalProvider
- Use cases untuk CompositionLocal
- Perbedaan dengan state hoisting
- Prop drilling problems
- Dependency provision dalam Compose
- Context API equivalent dalam Compose

## Waktu
2 hari

## Prioritas
⭐⭐ (Dependency provision)

## Catatan
CompositionLocal adalah alternatif yang baik untuk menghindari prop drilling dalam beberapa kasus.
