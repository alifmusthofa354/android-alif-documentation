# JSON Serialization

## Deskripsi
Serialisasi JSON adalah proses mengkonversi objek ke dan dari format JSON. Anda akan belajar menggunakan Kotlinx Serialization untuk efisiensi dan kompatibilitas dengan Kotlin.

## Tujuan Pembelajaran
- Menggunakan Kotlinx Serialization
- Mengkonversi objek ke/from JSON
- Memahami perbedaan dengan library lain

## Materi
- Konfigurasi Kotlinx Serialization
- @Serializable annotation
- Konversi objek ke JSON
- Konversi JSON ke objek
- Custom naming dengan @SerialName
- Perbedaan dengan Moshi
- Data parsing yang efisien

3 hari

## Prioritas
⭐⭐⭐ (Data parsing)

## Catatan
Kotlinx Serialization adalah pendekatan modern yang optimal untuk Kotlin dan Compose.
