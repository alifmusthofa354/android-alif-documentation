# Notifications

## Deskripsi
Notifications adalah cara penting untuk berinteraksi dengan pengguna. Anda akan belajar NotificationCompat, Channels, dan POST_NOTIFICATIONS permission untuk Android modern.

## Tujuan Pembelajaran
- Menggunakan NotificationCompat untuk backward compatibility
- Mengelola Notification Channels
- Menangani POST_NOTIFICATIONS permission

## Materi
- NotificationCompat dan backward compatibility
- Notification Channels (Android 8+)
- POST_NOTIFICATIONS permission (Android 13+)
- Notification types dan best practices
- Notification styling dan actions
- Old notification API problems
- User engagement melalui notifications
- Android 13+ permission requirements
