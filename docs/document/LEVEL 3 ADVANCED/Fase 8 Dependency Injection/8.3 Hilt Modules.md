# Hilt Modules

## Deskripsi
Hilt Modules digunakan untuk memberi tahu Hilt bagaimana cara menyediakan instans dependensi tertentu. Anda akan belajar berbagai cara untuk menyediakan dependensi.

## Tujuan Pembelajaran
- Menggunakan @Module dan @Provides dalam Hilt
- Menggunakan @Binds untuk binding interface
- Menggantikan manual factory pattern

## Materi
- @Module dalam Hilt
- @Provides untuk menyediakan dependensi
- @Binds untuk binding interface ke implementasinya
- Manual factory pattern problems
- Dependency provision dalam Hilt
- Provide dependencies dengan annotations
- Best practices dalam Hilt modules

## Catatan
Hilt Modules adalah cara utama untuk memberi tahu Hilt bagaimana cara membuat dependensi.
