# Debugging

## Deskripsi
Debugging adalah keterampilan penting dalam pengembangan aplikasi. Anda akan belajar menggunakan berbagai alat debugging yang tersedia di Android Studio.

## Tujuan Pembelajaran
- Menggunakan Logcat untuk melihat log aplikasi
- Menggunakan breakpoints untuk debugging
- Menggunakan Android Profiler

## Materi
- Penggunaan Logcat
- Log levels (verbose, debug, info, warn, error)
- Menyaring log
- Breakpoints dan debugging step-by-step
- Android Profiler (CPU, Memory, Network, Energy)
- Exception handling
- Troubleshooting umum

## Waktu
2 hari

## Prioritas
⭐⭐ (Essential skill)

## Catatan
Kemampuan debugging akan sangat mempercepat proses development dan troubleshooting.
