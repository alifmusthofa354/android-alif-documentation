# Lists & Grids

## Deskripsi
Membuat list dan grid yang efisien sangat penting dalam aplikasi. Jetpack Compose menyediakan Lazy composables yang efisien secara performa.

## Tujuan Pembelajaran
- Menggunakan LazyColumn dan LazyRow untuk list
- Menggunakan LazyVerticalGrid untuk grid
- Memahami efisiensi rendering dalam Compose

## Materi
- LazyColumn sebagai pengganti RecyclerView
- LazyRow untuk list horizontal
- LazyVerticalGrid untuk layout grid
- Items dan item dalam Lazy composables
- DiffUtil equivalent dalam Compose
- Performance considerations

## Waktu
3 hari

## Prioritas
⭐⭐⭐ (Efficient lists)

## Catatan
Lazy composables adalah pendekatan modern untuk membuat list yang efisien dan performa tinggi.
