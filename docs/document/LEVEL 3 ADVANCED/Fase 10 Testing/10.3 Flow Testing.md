# Flow Testing

## Deskripsi
Testing untuk Flow memerlukan pendekatan khusus karena nature-nya yang reactive. Anda akan belajar Turbine untuk testing reactive code.

## Tujuan Pembelajaran
- Menggunakan Turbine untuk testing Flow
- Memahami pendekatan testing untuk reactive code
- Menangani testing Flow dan StateFlow

## Materi
- Turbine library untuk Flow testing
- TestFlow dan ConsumeEach
- Testing StateFlow dan SharedFlow
- Collecting dan asserting dalam Flow testing
- Test suspend functions yang mengembalikan Flow
- Reactive code testing
- Flow testing best practices

## Waktu
3 hari

## Prioritas
⭐⭐ (Test reactive code)

## Catatan
Turbine menyediakan cara yang efisien untuk testing Flow dan reactive stream dalam aplikasi.
