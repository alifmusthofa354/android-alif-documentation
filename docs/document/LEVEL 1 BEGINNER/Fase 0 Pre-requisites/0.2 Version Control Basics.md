# Version Control Basics

## Deskripsi
Git adalah alat wajib bagi developer modern. Pada tahap ini, Anda akan belajar konsep dasar version control dan perintah Git yang paling sering digunakan.

## Tujuan Pembelajaran
- Memahami konsep version control
- Mampu menggunakan perintah Git dasar seperti clone, commit, push, dan pull
- Memahami workflow kolaborasi menggunakan Git

## Materi
- Apa itu version control
- Instalasi Git
- Konfigurasi Git (user.name, user.email)
- Git init, clone
- Git add, commit
- Git push, pull
- Git status, log
- Konsep remote repository

## Waktu
2 hari

## Prioritas
⭐⭐⭐ (Essential skill)

## Catatan
Kemampuan ini sangat penting untuk kolaborasi dan manajemen kode sumber.
