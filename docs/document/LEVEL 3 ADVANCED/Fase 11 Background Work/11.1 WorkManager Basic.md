# WorkManager Basic

## Deskripsi
WorkManager adalah library untuk menjalankan background tasks secara andal. Anda akan belajar OneTime dan Periodic work dalam konteks Android modern.

## Tujuan Pembelajaran
- Menggunakan WorkManager untuk background tasks
- Mengimplementasikan OneTime dan Periodic work
- Menggantikan pendekatan JobScheduler dan AlarmManager

## Materi
- Konsep WorkManager
- OneTimeWorkRequest
- PeriodicWorkRequest
- WorkManager vs pendekatan lama
- Constraints dalam WorkManager
- Background tasks management
- Modern approach untuk background work

## Waktu
4 hari

## Prioritas
⭐⭐⭐ (Background tasks)

## Catatan
WorkManager adalah pendekatan modern dan rekomendasi Google untuk background tasks.
