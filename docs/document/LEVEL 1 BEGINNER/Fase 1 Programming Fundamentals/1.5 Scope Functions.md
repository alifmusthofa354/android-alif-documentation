# Scope Functions

## Deskripsi
Scope functions adalah fitur unik Kotlin yang memungkinkan Anda membuat kode lebih ringkas dan ekspresif. Anda akan belajar berbagai scope functions dan kapan menggunakannya.

## Tujuan Pembelajaran
- Memahami berbagai scope functions
- Mengetahui perbedaan penggunaan masing-masing scope function
- Menggunakan scope functions dengan baik dalam kode

## Materi
- let
- run
- with
- apply
- also
- Perbedaan penggunaan dan return value
- Best practices dalam penggunaan scope functions
