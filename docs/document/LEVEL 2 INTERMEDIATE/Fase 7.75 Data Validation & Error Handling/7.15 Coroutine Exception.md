# Coroutine Exception

## Deskripsi
Exception dalam coroutine memiliki karakteristik khusus. Anda akan belajar CoroutineExceptionHandler dan supervisorScope lebih dalam untuk penanganan async exception.

## Tujuan Pembelajaran
- Menggunakan CoroutineExceptionHandler dalam konteks async
- Menggunakan supervisorScope secara mendalam
- Menghindari swallowed exceptions dalam async operations

## Materi
- CoroutineExceptionHandler dalam async context
- SupervisorScope deep dive
- Exception handling dalam nested coroutines
- Propagation dan penanganan exception
- Swallowed exceptions prevention
- Async error handling strategies
- Coroutine exception management
